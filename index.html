Minhas Skills

html5

Html5 é a linguagem de marcação para apresentar e estruturar o conteúdo web. Tenho experiência avançada, seguindo toda a semântica necessária para um bom funcionamento.

css3

Css3 é onde definimos a estilização de um projeto web. Tenho experiência avançada, mas sempre tem algo novo a se aprender com o css3.

sass

Sass é nada menos que um css turbinado, é um pré processador que tem várias funções e recursos que o css não tem. Tenho experiência avançada.

javascript

JavaScript é uma linguagem de programação muito utilizada no front-end, com ele conseguimos deixar o site mais dinâmico, entre vários outros recursos. Tenho experiência básica/intermediária.

php

Php é uma linguagem de programação muito utilizada no back-end, com ele conseguimos deixar o site mais dinâmico, coletar dados de formulário, enviar e receber cookies, entre outros recursos. Tenho experiência básica/intermediária.

git

Git é um sistema de controle de versão de arquivos. Permite que várias pessoas trabalhem no mesmo repositório sem que haja conflito. Tenho experiência intermediária/avançada.

wordpress

O WordPress é, basicamente, um CMS (Content Management System), onde gerenciamos todo o conteúdo de um site com seus incriveis plugins e outros recursos super importantes. Tenho experiência intermediária/avançada.

gulp

Gulp é uma ferramenta de automação de tarefas em JavaScript. Tarefas como minificar, otimizar e compilar arquivos. Tenho experiência intermediária/avançada.

filezilla

Filezilla é um cliente ftp muito utilizado para transferir sites, arquivos do seu computador para um servidor de hospedagem. Tenho experiência intermediária/avançada.
